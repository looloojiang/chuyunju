# Chuyunju H5 Demo

轻量的移动网页模板，包含主题切换与分享按钮。

快速预览：

1. 使用 Python 内置服务器（需要 Python 3）：

```bash
cd chuyunju-h5
python3 -m http.server 8000
# 在浏览器打开 http://localhost:8000
```

2. 或使用 `serve`（Node.js）：

```bash
npm install -g serve
cd chuyunju-h5
serve -p 5000
# 在浏览器打开 http://localhost:5000
```

文件列表：
- `index.html` - 入口页面
- `styles.css` - 样式
- `app.js` - 简单交互脚本
